/*
Group by clause : Group rows that have same value into summary rows. It collects data from multiple records and groups the result by one or more column.
Group by is generally used with aggregate funcions.
SELECT col1, col2 FROM table_name GROUP BY col_name(s);
Only those column could be selected which has been used in group by command else aggregate function could only be used with the group by function

Having clause
Similar to where it is used to appy conditions but after grouping   
*/

GENERAL ORDER : SELECT column(s) FROM table_name WHERE condition GROUP BY column(s)  HAVING condition ORDER BY column(s) ASC.
